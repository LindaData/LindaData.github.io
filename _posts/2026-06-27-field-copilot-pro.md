Field Copilot Pro is a mobile-friendly HVAC field-service prototype for Caloosa Cooling. It gives technicians and owners a working demo of daily job flow, equipment profiles, guided diagnostics, customer approval, service reporting, and English/Spanish support.

The current version is a Codex-managed GitHub baseline. It preserves the demo behavior from the original Lovable prototype while making the source repository and GitHub Pages demo independently maintainable.
