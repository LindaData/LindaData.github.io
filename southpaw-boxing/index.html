First class flow

Remove the fear before they walk in.

Elite gym sites do not just say “book now.” They tell first-timers exactly what happens, who they will meet, what to bring, and why they will not look stupid on day one.

1

Choose your lane

Beginner boxing, skills work, fight camp, youth, women’s classes, or private coaching. The user should self-identify fast.

2

Reserve in WellnessLiving

Use the existing account, payment, reminder, cancellation, and app ecosystem instead of rebuilding operations too early.

3

Arrive confident

Show parking, wraps/gloves expectations, class length, intensity, and “what if I’m out of shape?” answers before checkout.

Mission proof

Coach credibility

Jamie is the brand moat.

The site should lead with the rare mix: champion résumé, approachable coaching, and a family-room culture.

Content strategy

Make the origin story easy to scan.

Jamie’s achievements are real conversion assets. The rebuild should convert the long bio into proof blocks: boxing credentials, coaching range, comeback story, founder mission, and media highlights.

  • Use a short “why Jamie” section near the top, not only in a dropdown page.
  • Move media/news into proof cards and add a “watch Jamie” section.
  • Keep the approachable tone. This gym wins because it feels like family, not because it feels corporate.

Backend decision

Do not rebuild scheduling yet.

WellnessLiving is already handling account creation, reservations, reminders, cancellation links, payments, membership renewals, and the branded app. The new site should make that system feel premium through better calls-to-action and explanation.

WellnessLiving signup Google Play app

Architecture for v1

Static front door, SaaS operations, review layer.

  • Frontend: GitHub Pages static site first; fast, cheap, mobile-friendly.
  • Scheduling: Keep WellnessLiving links and app links.
  • Feedback: Founder review layer stores locally and can post to a secure Cloudflare Worker endpoint.
  • Analytics later: Track CTA clicks, first-class interest, program clicks, and review feedback themes.

Agent review board

What each specialist should pressure-test next.

This mirrors the CoPilot Field Pro review style: owner-friendly feedback now, structured enough for Codex/agents later.

UX agent

Test first-timer path, thumb reach, sticky CTAs, anxiety-reducing copy, and form clarity.

Architecture agent

Keep the site static until data requirements justify a backend. Avoid exposing any operational secrets.

Data agent

Define CTA events, source attribution, lead quality fields, class category clicks, and review-note schema.

SEO agent

Build local pages for Pineville/Charlotte boxing, women’s boxing, youth boxing, personal training, and fight camp.

Content agent

Rewrite pages into short sections: promise, proof, who it is for, what to expect, book.

Safety/legal agent

Add health disclaimers, privacy links, minors policy, cancellation handoff, and clear demo/not-official status until approved.
